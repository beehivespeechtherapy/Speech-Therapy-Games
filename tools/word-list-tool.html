Word list tool

Paste CSV from Google Sheets (or upload a file). Two layouts supported:
Transposed (columns = sets): Row 1 = setID, row 2 = setLabel, prompt, targetSound, contrastSound, folder, then word1, word2, word3… Each column (after the first) is one word set. You can repeat this block below (same row labels) to add more sets when you run out of columns (e.g. past column Z).
Row-per-pair: Header row with setId, setLabel, prompt, targetSound, contrastSound, folder, word1, word2 — then one data row per pair. Then download word-sets.json and save as word-lists/word-sets.json. Full format details.

# Map Background Images

Add themed background images here for your game maps.

## Recommended Themes

- **jungle-bg.jpg** - Jungle/rainforest theme
- **castle-bg.jpg** - Castle/medieval theme
- **space-bg.jpg** - Space/galaxy theme
- **ocean-bg.jpg** - Underwater/ocean theme
- **desert-bg.jpg** - Desert/safari theme
- **winter-bg.jpg** - Snow/winter theme

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 1200x600px recommended (2:1 ratio)
- **Size**: 200-500 KB (compressed)
- **Style**: Illustrated, colorful, child-friendly

## Free Image Sources

- **Pixabay**: https://pixabay.com/illustrations/search/landscape/
- **Unsplash**: https://unsplash.com/s/photos/cartoon-landscape
- **Freepik**: https://www.freepik.com/ (free with attribution)
- **OpenGameArt**: https://opengameart.org/

## Usage

Reference in your game's config.json:

```json
{
  "map": {
    "theme": "jungle",
    "backgroundImage": "../../assets/maps/jungle-bg.jpg"
  }
}
```
